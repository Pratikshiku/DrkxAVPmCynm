### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。

### 三、功能描述
系统分为（身份）：普通用户（学生）、学校、银行、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 助学贷款
6. 我申请贷款
7. 放贷信息
8. 还贷信息
9. 公告资讯




**学校功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 助学贷款管理
6. 申请贷款管理
7. 公告资讯





**银行功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 助学贷款管理
6. 申请贷款管理
7. 放贷信息管理
8. 还贷信息管理
9. 公告资讯




**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 学生管理
4. 学校管理
5. 银行管理
6. 助学贷款管理
7. 申请贷款管理
8. 放贷信息管理
9. 还贷信息管理
10. 系统管理（公告资讯）
